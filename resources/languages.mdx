## Supported Languages

The Spectropic API supports a wide range of languages for transcription. The following table lists the languages supported by the API.

Use the `language` field in the request body to specify the language of the media file you want to transcribe. Leave the field empty to use automatic language detection. 
<Tip>
Want to use a fine tuned model for a specific language? Check the table below to see which languages have fine tuned models available. **You must specify the language code in the `language` field** to use a fine tuned model.
</Tip>
