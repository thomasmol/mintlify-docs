## Welcome

Welcome to the Spectropic documentation. This documentation is intended for developers who want to use the Spectropic API to build applications. 

Generate an API key by signing up at [https://spectropic.ai](https://spectropic.ai).

## Authentication

All API endpoints are authenticated using your API Key as a Bearer token.

```bash Request Example
  curl  --request GET \
    --url https://api.spectropic.ai/v1/transcripts \
    --header: 'Authorization: Bearer <apikey>' \
    --header: 'Content-Type: application/json'
```
