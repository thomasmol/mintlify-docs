---
title: 'Find Transcript'
api: 'GET /v1/transcripts/{id}'
---

Get a single transcript by id. If transcript is not ready yet, the `segments` field will be empty.

<ParamField path="id" type="uuid" >
  The id of the transcript
</ParamField>

<ResponseExample>

```json Response
{ 
  "transcriptId": "someuuid",
  "segments": [
    {
      "startTime": 0,
      "endTime": 1.5,
      "speaker": "speaker1",
      "text": "Hello, my name is John Doe",
      "words": [
        {
          "start": 0,
          "end": 0.2,
          "word": "Hello,"
        },
        ...
      ]
    },
    {
      "startTime": 1.5,
      "endTime": 2.5,
      "speaker": "speaker2",
      "text": "Hello, my name is Jane Doe",
      "words": [
        {
          "start": 1.5,
          "end": 1.7,
          "word": "Hello,"
        },
        ...
      ]
    }
  ]
}
```
</ResponseExample>
