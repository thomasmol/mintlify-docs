---
title: 'Create Transcript'
api: 'POST /v1/transcripts'
---

Create a new transcript. Returns a transcript ID that can be used to check the status of the transcript.

## Request
### Body

<ParamField body="url" type="string">
    Url of the audio file to be transcribed.
</ParamField>

{/* OR

<ParamField body="file" type="file">
    Media file to be transcribed.
</ParamField> */}

<ParamField body="numSpeakers" type="integer">
    Number of speakers in the audio file. If not provided, the number of speakers is automatically detected.
</ParamField>

<ParamField body="language" type="string">
    Language of the audio file. If not provided, the language is automatically detected.
</ParamField>

<ParamField body="vocabulary" type="string">
    Vocabulary to be used for the transcription. Provide names, acronyms, and other words that are specific to your domain.
</ParamField>

<ParamField body="webhook" type="string">
    URL to be called when the transcript is ready.
</ParamField>

## Response

<ResponseField name="transcriptId" type="string">
    ID of the transcript.
</ResponseField>

<ResponseField name="status" type="string">
    Status of the transcript. Possible values are `started`.
</ResponseField>
