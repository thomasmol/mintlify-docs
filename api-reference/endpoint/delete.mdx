---
title: 'Delete Transcript'
api: 'DELETE /v1/transcripts/{id}'
---

Get all transcripts, sorted by date descending. If a transcript is not yet complete, the `segments` attribute will be empty.

## Request
No parameters required.

## Response

Returns an array of `transcript` objects.

<ResponseExample>

```json Response
{ 
  "transcripts": [
    {
        ...
    }
  ]
}
```
</ResponseExample>
