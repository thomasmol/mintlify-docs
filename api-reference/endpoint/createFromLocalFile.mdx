---
title: 'Create Transcript (local file)'
api: 'POST /v1/transcripts'
---

Create a new transcript from a local file. 
Send file as a multipart/form-data request.
Currently the maximum file size is 100MB.

Returns a transcript ID that can be used to check the status of the transcript.

## Request
### Form Data

<ParamField header="Content-Type" default="multipart/form-data" type="string">
    multipart/form-data
</ParamField> 

<ParamField header="file" type="file">
    The audio file to be transcribed.
</ParamField>


## Response

<ResponseField name="transcriptId" type="string">
    ID of the transcript.
</ResponseField>

<ResponseField name="status" type="string">
    Status of the transcript. Possible values are `started`.
</ResponseField>
